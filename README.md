## Sobre o Projeto

Esse projeto inicial visa mostrar meus conhecimentos com a linguagem **Go**. É o projeto de uma lanchonete, com funções de **Menu**, **Pedido** e **Extrato**.

Está sendo utilizado:
- **Structs**
- **Funções**
- **Packages** da linguagem

Estarei realizando alguns aprimoramentos à medida que aprofundo meu conhecimento na linguagem.
